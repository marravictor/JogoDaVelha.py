# Jogo da Velha em Python 

## Descrição
Este projeto consiste no desenvolvimento de um Jogo da Velha criado inicialmente no Flowgorithm, utilizando fluxogramas para representar toda a lógica do programa. Após a conclusão do algoritmo, o projeto foi convertido para a linguagem Python e desenvolvido no Visual Studio Code (VS Code).

O objetivo foi aplicar conceitos de lógica de programação, estruturas condicionais, laços de repetição, matrizes e validação de dados, transformando um fluxograma em uma aplicação funcional.

## Funcionalidades
- Tabuleiro 3x3.
- Dois jogadores (X e O).
- Alternância automática entre os jogadores.
- Verificação de vitória em linhas, colunas e diagonais.
- Verificação de empate.
- Exibição do tabuleiro após cada jogada.
- Validação das posições escolhidas.

## Tecnologias Utilizadas
- Flowgorithm
- Python
- Visual Studio Code (VS Code)

## Conceitos Aplicados
- Algoritmos
- Fluxogramas
- Estruturas condicionais (if, elif e else)
- Estruturas de repetição (while)
- Entrada e saída de dados
- Validação de informações

## Objetivo do Projeto
O projeto foi desenvolvido com a finalidade de praticar a lógica de programação e compreender o processo de conversão de algoritmos representados por fluxogramas para uma linguagem de programação real, neste caso o Python.
